# Contributing Config Changes

This project accepts config-only contributions for Deviations, Traits, and Variants.

Please only change config data:
- `config.js`

Do not include UI/app code changes (`index.html`, styles, or app logic) in config PRs.

## Browser-Only GitHub Flow
1. Open the **Edit Config on GitHub** link in the app.
2. If needed, GitHub will automatically create your fork.
3. Make your config edits in the browser editor.
4. Click **Commit changes**.
5. Click **Propose changes** and open a pull request.
6. Set PR base branch to `dev` before submitting.

## Tips
- Keep edits focused and small.
- Preserve existing data shape and formatting style.
- Add clear PR notes about what was added/updated.
