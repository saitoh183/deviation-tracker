# Contributing Your Configuration

You can contribute new **Deviations, Traits, or Variants** to the tracker by submitting your configuration file.

## 1. Export your configuration
In the app:

1. Open **Settings**
2. Click **Save Config**

This will download a JSON file containing your current configuration.

---

## 2. Rename the file

Rename the downloaded file using this format:
(name)-config.json

Examples:
- mylatest-config.json
- player123-config.json

---

## 3. Upload the file

Upload your file to here

Then create a **Pull Request targeting the `dev` branch**.

---

## Notes

- Only upload files exported using **Save Config**
- Do **not edit `config.js` directly**
- Your submission will be reviewed before being merged
